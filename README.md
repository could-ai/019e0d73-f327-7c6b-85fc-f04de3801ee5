# Daily Tasks App

A simple Flutter app to manage your daily tasks. It allows you to add tasks, mark them as complete, and delete them. This serves as a foundation for tracking your daily work and typing out your tasks.

## Features
- Add daily tasks via text input.
- Check off tasks as completed (with strike-through styling).
- Delete tasks.

## Tech Stack
- Flutter
- Dart

## Setup
1. Ensure you have Flutter installed.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to launch the app on your preferred device.
